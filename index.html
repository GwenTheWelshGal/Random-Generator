Random Generator

Welcome. On this page, you will find a simplistic random generator that I have coded in Python. I trust that you will find it useful.

To use it, enter a list of items with each list separated by a semicolon - ; - in the boxes on the left and the right, and then click the Start button. Make sure that both lists have the same number of items, or it will not work.
