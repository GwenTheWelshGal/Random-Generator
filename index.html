Random Generator

Hi. On this page, you will find a simplistic random generator that I have put together using a combination of Python and HTML, and I hope that you will find it useful in some capacity. To use it, enter a list of items in the text box on the left and an equally large list in the text box on the right. If the lists are not the same length, then it will not work. Once you've done so, click Start. Below, a list of outputs will be displayed.

Keep in mind, however, that this is very much a work in progress.
